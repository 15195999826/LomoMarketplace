# InkMon 生态关系

## 栖息地类型

| 栖息地 | 英文 | 常见属性 | 环境特点 |
|-------|------|---------|---------|
| 森林 | Forest | 草、虫 | 郁郁葱葱，树木茂密 |
| 火山 | Volcano | 火、岩 | 高温环境，熔岩流动 |
| 海洋 | Ocean | 水 | 深海浅海，水生生物 |
| 湖泊 | Lake | 水 | 淡水环境，水草丰富 |
| 山脉 | Mountain | 岩、飞 | 高海拔，峭壁悬崖 |
| 洞穴 | Cave | 暗、岩 | 阴暗潮湿，地下环境 |
| 沙漠 | Desert | 地、火 | 干燥炎热，昼夜温差大 |
| 冰原 | Tundra | 冰 | 极寒地带，积雪覆盖 |
| 平原 | Plains | 草、地 | 开阔草地，温和气候 |
| 沼泽 | Swamp | 水、毒 | 潮湿泥泞，植被腐烂 |
| 天空 | Sky | 飞、龙 | 高空区域，云层之间 |
| 城市 | Urban | 钢、电 | 人类聚居地 |
| 神殿 | Temple | 光 | 神圣建筑，古老遗迹 |
| 深渊 | Abyss | 暗、龙 | 神秘深处，少有人至 |

## 生态关系类型

| 关系 | 英文 | 描述 | 设计意义 |
|-----|------|------|---------|
| 捕食者 | predator | 捕食某些 InkMon | 食物链顶端 |
| 猎物 | prey | 被某些 InkMon 捕食 | 食物链底层 |
| 共生 | symbiosis | 互利共存 | 合作关系 |
| 竞争 | competition | 争夺资源 | 生态位重叠 |

## 食性类型

| 类型 | 英文 | 描述 | 典型 InkMon |
|-----|------|------|------------|
| 草食 | herbivore | 以植物为食 | 草属性、温顺型 |
| 肉食 | carnivore | 以其他 InkMon 为食 | 强攻型、猎手型 |
| 杂食 | omnivore | 植物和肉类都吃 | 适应性强的类型 |
| 特殊 | special | 吸收能量/矿物等 | 光/暗/钢属性 |

## 生态关系设计原则

### 自然逻辑
- 属性克制可以反映在生态关系中
- 火属性可能是虫属性的天敌
- 水属性可能是火属性的威胁

### 食物链层级
```
传说级 InkMon (顶级)
       ↑
  完全体 InkMon
       ↑
  中间形态 InkMon
       ↑
  幼年 InkMon / 虫类
       ↑
    植物 / 矿物
```

### 栖息地匹配

| 属性 | 推荐栖息地 |
|-----|-----------|
| 火 | 火山、沙漠 |
| 水 | 海洋、湖泊、沼泽 |
| 草 | 森林、平原 |
| 电 | 山脉、城市 |
| 冰 | 冰原、山脉高处 |
| 岩 | 山脉、洞穴 |
| 地 | 沙漠、平原、洞穴 |
| 飞 | 天空、山脉 |
| 虫 | 森林、沼泽 |
| 毒 | 沼泽、森林深处 |
| 暗 | 洞穴、深渊 |
| 光 | 神殿、天空 |
| 钢 | 城市、洞穴(矿区) |
| 龙 | 天空、深渊、各种极端环境 |

## 生态关系示例

### 火焰猴的生态

```json
{
  "ecology": {
    "habitat": "火山地带",
    "diet": "omnivore",
    "predators": ["岩龙"],
    "prey": ["火山虫", "岩蜥"],
    "symbiosis": ["火山鸟"],
    "competition": ["熔岩熊"]
  }
}
```

**解读**:
- 栖息在火山地带（符合火属性）
- 杂食性（符合猴类特征）
- 被岩龙捕食（岩克火）
- 捕食虫类和小型蜥蜴
- 与火山鸟共生（互相警示危险）
- 与熔岩熊竞争食物资源

### 冰狐的生态


## JSON 格式

```json
{
  "ecology": {
    "habitat": "火山地带",
    "diet": "omnivore",
    "predators": ["岩龙"],
    "prey": ["虫类 InkMon"]
  }
}
```

**字段说明**:
| 字段 | 类型 | 说明 |
|-----|------|------|
| habitat | string | 主要栖息地 |
| diet | string | 食性 (herbivore/carnivore/omnivore/special) |
| predators | string[] | 天敌列表 |
| prey | string[] | 猎物列表 |

## 设计建议

### 增加世界观深度
- 描述 InkMon 在生态系统中的角色
- 考虑与其他 InkMon 的互动关系
- 让玩家感受到世界的真实性

### 保持合理性
- 属性弱点可以成为天敌
- 体型差异影响食物链地位
- 栖息地应与属性匹配

### 故事潜力
- 生态关系可以衍生剧情
- 共生关系可以设计特殊技能
- 竞争关系可以设计对战任务

- [ ] 生态位是否有特色？
