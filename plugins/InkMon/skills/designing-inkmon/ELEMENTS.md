## 克制规则

采用类宝可梦属性克制逻辑，Claude 已熟知基础规则。

### InkWorld 特有规则

| 规则 | 说明 |
|-----|------|
| 光 ↔ 暗 | 互相克制 (2x) |
| 虫 → 光 | 虫克制光 (2x) |
| 钢免疫毒 | 毒对钢完全无效 (0x) |
| 地免疫电 | 电对地完全无效 (0x) |
| 飞免疫地 | 地对飞完全无效 (0x) |

### 效果倍率

| 倍率 | 显示文本 |
|-----|---------|
| 2x | "超级有效!" |
| 1x | (无提示) |
| 0.5x | "效果不太好..." |
| 0x | "完全无效!" |

## 双属性计算

双属性时，伤害倍率**相乘**：

| 情况 | 示例 | 结果 |
|-----|------|------|
| 4倍弱点 | 火 vs 草/虫 | 2×2 = 4x |
| 抵消 | 火 vs 火/水 | 0.5×2 = 1x |
| 免疫优先 | 电 vs 地/飞 | 0x (地免疫电) |

## 属性选择建议

### 强力组合
| 组合 | 优点 |
|-----|------|
| 钢/龙 | 抗性极多 |
| 水/地 | 仅弱草4x |
| 飞/钢 | 免疫地，抗性多 |

### 需谨慎的组合
| 组合 | 缺点 |
|-----|------|
| 冰/岩 | 弱点太多 |
| 草/冰 | 4倍弱火 |
| 虫/草 | 4倍弱火、飞 |
